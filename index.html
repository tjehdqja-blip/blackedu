🔗 구글 드라이브 연결 설정
구글 드라이브에 문제 데이터를 저장하면, 학원 내 모든 선생님이 같은 데이터를 실시간으로 공유할 수 있습니다.
처음 한 번만 설정하면 이후에는 자동으로 연결됩니다.
STEP 1 — Google Cloud 설정 (관리자 1회)
1Google Cloud Console (console.cloud.google.com) 접속 → 새 프로젝트 생성
2왼쪽 메뉴 → API 및 서비스 → 라이브러리 → Google Drive API 검색 → 사용 설정
3사용자 인증 정보 → OAuth 2.0 클라이언트 ID 만들기 → 유형: 웹 애플리케이션
4승인된 JavaScript 원본에 이 앱이 열리는 주소 추가 (예: https://claude.ai)
5생성된 클라이언트 ID 복사
STEP 2 — 공유 파일 ID (선택사항)
처음 연결 시 자동으로 Drive에 tomahawk_problems.json 파일이 생성됩니다.
여러 선생님이 같은 파일을 공유하려면: 관리자가 파일 ID를 복사해서 다른 선생님들에게 전달하세요.
파일 ID를 모르면 나중에 설정하거나, 로컬 저장으로 임시 사용
